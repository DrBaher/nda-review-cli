nda-review-cli

GitHub README Getting Started

Local-first NDA review, drafting, and two-party negotiation.

Try the three headline flows below in a sandboxed session. Nothing you enter is logged; sandboxes auto-expire after 30 minutes. For real use, install locally: pipx install nda-review-cli.

Demo runs the deterministic rules engine only — no LLM provider is wired up here. The --llm, --agent, and counterparty-profile features work the same way locally; install the CLI to use them.
